This repository contains a custom STM32 bare-metal peripheral driver framework developed without relying on high-level libraries. The project focuses on low-level embedded firmware development using direct register manipulation and structured driver architecture.

Implemented peripheral modules include:

"GPIO driver (pin configuration, input/output modes)"
"I2C driver (master communication, sensor interfacing)"
"SPI driver (full-duplex communication, peripheral interface)"
"USART driver (serial communication and debugging)"

The goal of this project is to understand STM32 hardware architecture, peripheral initialization, communication protocols, and embedded software design principles by building reusable and scalable drivers from scratch.
